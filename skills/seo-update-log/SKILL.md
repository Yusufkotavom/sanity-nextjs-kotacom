# Skill: SEO Update Log Maintenance

## Purpose
Enforce consistent changelog updates in `docs/seo-updates.md` for every repository modification.

## When to Use
- Any code/config/docs change in this repository.

## Required Actions
1. Open `docs/seo-updates.md`.
2. Append a new entry using the template.
3. Include changed file paths and SEO impact (or `No direct SEO impact`).
4. Add verification results.

## Completion Criteria
- Task is incomplete if no new log entry is added.
