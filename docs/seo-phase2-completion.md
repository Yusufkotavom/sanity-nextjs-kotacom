## 📝 Customization (Opsional)

Anda bisa customize di Sanity Studio:

### 1. Ubah Judul/Deskripsi Block
- Buka halaman di Studio
- Klik block yang ingin diedit
- Edit field "Title" dan "Description"
- Save

### 2. Ubah Urutan Block
- Buka halaman di Studio
- Drag & drop blocks untuk mengubah urutan
- Save

### 3. Ubah Warna Background
- Buka halaman di Studio
- Klik block yang ingin diedit
- Pilih "Color Variant": background, muted, accent, dll
- Save

### 4. Ubah Padding
- Buka halaman di Studio
- Klik block yang ingin diedit
- Toggle "Top Padding" dan "Bottom Padding"
- Save
